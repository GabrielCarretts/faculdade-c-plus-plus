# College C++ Projects

## Description

This repository contains a collection of C++ console applications developed during my first semester of college.

Each project focuses on building a strong foundation in programming logic, gradually evolving from basic input/output operations to structured validation, decision-making with `if` statements, and more refined user interaction.

---

## Projects

* **Simple Calculator** – performs basic arithmetic operations with user-selected operators 
* **Temperature Converter** – converts between Celsius and Fahrenheit using conditional logic 
* **Car Cost Calculator** – computes final consumer price including taxes and markup 
* **Car Mileage Tracker** – calculates fuel consumption (KM/L) with validated input 
* **USD to BRL Converter** – converts currency using exchange rates 
* **Inflation/Deflation Calculator** – determines price variation and classifies it as inflation or deflation 
* **Point of Sale Calculator** – simulates a small checkout system with discounts 
* **Popsicle Revenue Tracker** – calculates total revenue across multiple product types 
* **Consortium Manager** – computes total paid amount and remaining debt with validation and loop control 
* **Employee Salary Calculator** – calculates net salary with multiple deductions 
* **Salesperson Salary Calculator** – calculates commission-based earnings 
* **Hotel Bill Estimator** – applies conditional service fees based on stay duration 
* **Package String Estimator** – calculates required string length for packaging using geometric reasoning 
* **Sphere Volume Tool** – calculates the volume of a sphere using a mathematical formula 
* **Days Lived Tracker** – computes approximate days lived from formatted date input 

---

## Concepts Practiced

* Variables and data types (`int`, `double`, `char`, `string`)
* Input and output (`cin`, `cout`, `getline`)
* Arithmetic operations and formulas
* Conditional logic (`if`, `else if`, `else`)
* Loops (`while`, `do-while`, `for`)
* Input validation and error handling
* Boolean logic (`&&`, `||`, `!=`, `==`)
* Output formatting (`<iomanip>`, `fixed`, `setprecision`)
* Basic UI/UX improvements using ANSI terminal colors
* Problem decomposition (input → process → output)

---

## How to Run

1. Navigate to a project folder:

   ```bash
   cd project-folder
   ```

2. Compile the program:

   ```bash
   g++ file_name.cpp -o app
   ```

3. Run the executable:

   ```bash
   ./app
   ```
